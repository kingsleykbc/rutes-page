THE RUTES PROJECT

The remote user evaluation and usability testing suite (RUTES) project is a system that allows web developers to create, monitor, and view results of user tests on their applications.

Get Started

Simple, Fast, Flexible

RUTES provides developers and testers with the speed and convenience of carrying out user studies by providing a platform that allows for test activities to be carried out. RUTES also provides additional features like element annotation and chat between developers and testers.
