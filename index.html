THE RUTES PROJECT

The remote user evaluation and usability testing suite (RUTES) project is a system that allows web developers to create, monitor, and view results of user tests on their applications.

Get Started

Simple, Fast, Flexible

RUTES provides developers and testers with the speed and convenience of carrying out user studies by providing a platform that allows for test activities to be carried out. RUTES also provides additional features like element annotation and chat between developers and testers.

Download presentation

GETTING STARTED

  • 1. Create Project and tests on the Admin Dashboard

    Sign up on the Rutes Admin app and create a new project. In the add form, specify the different test instructions, questionnaires, etc. and enter the emails of your approved testers. (NOTE: For exploration only, login with the default credentials by clicking the "Pre-fill test credentials button").

  • 2. Import tester library into web project

    Instructions for this are provided in the details section of the project on the dashboard with the 'embed code' snippets and the library's download link (NOTE: For exploration, visit the test website which has the library setup for the "Library test" project defined in the default credentials account in step 1 above ).

  • 3. Interact with testers and results

    Observe results from participants (UI annotations, test completions, questionnaire responses, etc.) and chat with participants during sessions. (NOTE: For exploration, view the results of the "Library test" project under the default credentials account. Also note that screen recording may be removed in the future due to cost.).
